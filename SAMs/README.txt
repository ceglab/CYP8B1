
This directory contains the SAM (Sequence Alignment Map) files generated by performing a blastn search against SRA (Short Read Archive) datasets. All blast search operations were done using the online tool available on the ncbi website (https://blast.ncbi.nlm.nih.gov/Blast.cgi?PROGRAM=blastn&PAGE_TYPE=BlastSearch&BLAST_SPEC=SRA&LINK_LOC=blasttab)
